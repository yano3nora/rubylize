rubylize
===
> https://deno.land/x/rubylize

# Installation & Update
```sh
$ deno install -f -n rubylize -A https://deno.land/x/rubylize/main.ts
```

# Usage
```sh
$ echo 'こんにちは、世界。' | rubylize -
# こんにちは、<ruby>世界<rt>せかい</rt></ruby>。
```
